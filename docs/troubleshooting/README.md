## Contributing Troubleshooting Guides

When adding new troubleshooting guides:

1. **Create detailed problem description** - What the user sees vs expected behavior
2. **Identify root causes** - Technical reasons for the issue  
3. **Provide step-by-step solutions** - Exact code changes and file locations
4. **Include verification steps** - How to test the fix
5. **Add to relevant indexes** - Update this README and main docs index
