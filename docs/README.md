## Project Overview

Collab Room is a mobile-first Web3 collaboration platform that revolutionizes professional networking through an intelligent, interactive matching ecosystem. It focuses on creating seamless, responsive professional connection experiences with cutting-edge blockchain technology.

### Key Features

- User and company profile management
- Collaboration creation and discovery
- Swipe-based matching interface
- Application system for collaborations
- Notification system
- Admin panel for user management
- Multi-blockchain network support
- Advanced filtering options
- Telegram integration for authentication

### Technology Stack

- Frontend: React, TypeScript, Shadcn/ui, Tailwind CSS, Framer Motion
- Backend: Node.js, Express
- Database: PostgreSQL with Drizzle ORM
- Authentication: Telegram WebApp
- Data Fetching: React Query
- Routing: Wouter
- Validation: Zod
- State Management: React Context and local state

## Getting Started

To run the application locally:

1. Clone the repository
2. Install dependencies with `npm install`
3. Set up the environment variables
4. Start the application with `npm run dev`
