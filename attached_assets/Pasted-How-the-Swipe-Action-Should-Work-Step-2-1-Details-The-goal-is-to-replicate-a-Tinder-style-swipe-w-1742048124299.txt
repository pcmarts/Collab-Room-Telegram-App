How the Swipe Action Should Work (Step 2.1 Details)

The goal is to replicate a Tinder-style swipe with three actions:
	•	Swipe Left → Pass
	•	Swipe Right → Request
	•	Tap → Flip Card (More Info)

🔹 Swipe Actions Breakdown

Each card should be draggable left and right using Framer Motion.

1. Swipe Left (Pass)
	•	User Action: Swipes the card to the left.
	•	Effect:
	•	Card animates off-screen to the left.
	•	Next card slides in smoothly.
	•	Card is removed from the stack.
	•	No further action taken (not saved in the database).
	•	Visual Feedback:
	•	Slight red glow when swiping left.
	•	Card fades out as it moves left.

2. Swipe Right (Request)
	•	User Action: Swipes the card to the right.
	•	Effect:
	•	Card animates off-screen to the right.
	•	Next card slides in smoothly.
	•	The collaboration request is saved (placeholder for now).
	•	Visual Feedback:
	•	Slight green glow when swiping right.
	•	A brief checkmark animation appears before the card disappears.

3. Tap (More Info / Flip Card)
	•	User Action: Taps the card.
	•	Effect:
	•	The card flips (animated rotation) to show detailed information.
	•	Tap again to flip back to the original view.
	•	Visual Feedback:
	•	Card rotates around the Y-axis for a smooth flip effect.

🔹 Technical Details for Swipe Implementation

1. Set Up the Swipe Gesture

Framer Motion provides drag for swipe interactions.
Example code for a motion card:

<motion.div
  drag="x"
  dragConstraints={{ left: 0, right: 0 }}
  onDragEnd={(event, info) => {
    if (info.offset.x < -100) {
      handlePass(); // Swipe Left
    } else if (info.offset.x > 100) {
      handleRequest(); // Swipe Right
    }
  }}
>
  {/* Card Content Here */}
</motion.div>

	•	drag="x" allows only horizontal movement.
	•	onDragEnd checks swipe distance:
	•	If info.offset.x < -100, treat as pass.
	•	If info.offset.x > 100, treat as request.

2. Flip Card on Tap

Use useState to track whether the card is flipped.

const [flipped, setFlipped] = useState(false);

Modify the card’s rotateY when tapped:

<motion.div
  animate={{ rotateY: flipped ? 180 : 0 }}
  transition={{ duration: 0.5 }}
  onClick={() => setFlipped(!flipped)}
>
  {flipped ? <DetailedInfo /> : <QuickView />}
</motion.div>

	•	When tapped, rotateY flips the card 180 degrees.
	•	The front side shows title + company, the back side shows detailed info.
