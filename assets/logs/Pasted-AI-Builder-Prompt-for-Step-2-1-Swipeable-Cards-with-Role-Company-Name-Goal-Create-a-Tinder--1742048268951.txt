🎯 Goal:
Create a Tinder-style swipe interaction for collaboration matching using Framer Motion, while displaying:
	•	User’s Role Title (e.g., VP of Marketing)
	•	Company Name
	•	Collaboration Type (e.g., Co-Marketing, Podcast Guest, etc.)

📌 AI Builder Instructions:

1. Implement Swipeable Card Layout
	•	Each card should display:
	•	User’s Role Title (e.g., VP of Marketing)
	•	Company Name (e.g., Blockchain Weekly)
	•	Collaboration Type (e.g., Podcast Guest)
	•	Design for one card at a time in the Discover section.

2. Enable Swipe Actions (Framer Motion)

Swipe Left (Pass)
	•	User Action: Swipes the card to the left.
	•	Effect:
	•	Card animates off-screen to the left.
	•	Next card slides in smoothly.
	•	Card is removed from the stack (not saved).
	•	Visual Feedback:
	•	Slight red glow when swiping left.
	•	Card fades out as it moves left.

Swipe Right (Request Collaboration)
	•	User Action: Swipes the card to the right.
	•	Effect:
	•	Card animates off-screen to the right.
	•	Next card slides in smoothly.
	•	The collaboration request is stored (placeholder for now).
	•	Visual Feedback:
	•	Slight green glow when swiping right.
	•	A brief checkmark animation appears before the card disappears.

Tap (More Info / Flip Card)
	•	User Action: Taps the card.
	•	Effect:
	•	The card flips (animated rotation) to show detailed information about the user and the collaboration.
	•	Tap again to flip back to the original view.
	•	Visual Feedback:
	•	Card rotates around the Y-axis for a smooth flip effect.
