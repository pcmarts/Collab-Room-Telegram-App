AI Design Brief: Onboarding Flow for Collab Room

Objective

Design an intuitive, step-by-step onboarding flow that captures all necessary user, company, and collaboration details. The flow should include smart input fields (e.g., pre-filled website URLs, file upload for logos, dropdowns for industry and collaboration types).

Step 1: User Information

Fields to capture:
	•	Full Name (First & Last Name)
	•	Telegram Handle (Prefilled with @)
	•	LinkedIn Profile URL (Prefilled with https://www.linkedin.com/in/)
	•	Bio (Short description about user)

Logic & UX Enhancements:
	•	Use placeholders (@username, https://www.linkedin.com/in/your-profile)
	•	Add character limits for bio (e.g., 200 characters)
	•	Autofill Telegram username if signed in via Telegram

Step 2: Company Information

Fields to capture:
	•	Company Name
	•	Website (Prefilled with https://)
	•	Logo Upload (File upload instead of URL input)
	•	Industry (Dropdown with Web3-specific options)
	•	Company LinkedIn URL (Prefilled with https://www.linkedin.com/company/)
	•	Company Twitter/X Handle (Prefilled with https://twitter.com/)
	•	Company Telegram Announcement/Chat URL (Optional)

Logic & UX Enhancements:
	•	Autofill fields based on domain verification where possible
	•	Logo upload should support PNG/JPG, with auto-cropping to square format
	•	Industry dropdown should be limited to Web3 categories (to be provided later)

Step 3: Collaboration Preferences

Fields to capture:
	•	Collaboration Type (Dropdown with predefined options)
	•	Collaboration Description (Brief details about what the user/company is looking for)
	•	Tags (Optional, allow multi-select or text input)

Logic & UX Enhancements:
	•	Dropdown with predefined Web3 collab types (to be provided later)
	•	Limit tags to relevant Web3 categories
	•	Show examples of good descriptions

Additional Considerations
	•	Progressive Form Flow (Each step leads smoothly into the next)
	•	Save & Resume Feature (If users drop off, they can resume from where they left)
	•	Mobile-First UI (Since many will sign up via Telegram mobile)
	•	Auto-validate URLs (Ensure proper formatting and domain validity)
	•	File Compression for Logos (Optimized image upload for faster performance)

Next Steps:

	•	Provide Collaboration Type Options for the collaboration dropdown
	•	Ensure the design follows minimalist and Web3-friendly aesthetics
